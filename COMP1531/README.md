# COMP1531 - Software Engineering Fundamentals

This directory contains projects and assignments from COMP1531: Software Engineering Fundamentals.

## Course Overview

COMP1531 introduces software engineering principles and practices, covering:
- Software design and architecture
- Object-oriented programming concepts
- Version control with Git
- Testing strategies and test-driven development
- Agile development methodologies
- Team collaboration and project management
- Code quality and documentation
